
# Get started

Terminal commands:


Run demo Accounts
========
```
npx hardhat node
```



Deploy accounts
======
Open a new terminal window.
```
npx hardhat run scripts/deploy.js --network localhost
```

**Should deploy to:**
0x5FbDB2315678afecb367f032d93F642f64180aa3



Run app/ localhost
========
```
npm run dev
```
